# 🎲 Würfelspiel

Ein einfaches Zwei-Spieler-Würfelspiel gebaut mit HTML, CSS und Vanilla JavaScript.

## 📸 Vorschau

<img width="1374" height="637" alt="image" src="https://github.com/user-attachments/assets/20fcc57b-6cc5-45ae-8000-3cf73c82e4de" />


Zwei Spieler werfen gleichzeitig einen Würfel — wer die höhere Zahl hat, gewinnt die Runde und bekommt einen Punkt. Der Punktestand wird über mehrere Runden gespeichert.

---

## 🚀 Features

- Würfelanimation mit Flicker-Effekt
- Gewinner/Verlierer-Hervorhebung pro Runde
- Punktestand über mehrere Runden
- Unentschieden-Erkennung
- Tastatursteuerung (`Leertaste` = würfeln, `R` = zurücksetzen)
- Responsives Design für Mobile

---

## 📁 Projektstruktur

```
wuerfelspiel/
├── index.html    ← Struktur & Markup
├── style.css     ← Styling & Animationen
├── script.js     ← Spiellogik
└── README.md
```

---

## 🛠️ Technologien

- HTML5
- CSS3 (Animationen, CSS Variables, Grid)
- JavaScript (Vanilla, kein Framework)

---

## ▶️ Starten

Einfach `index.html` im Browser öffnen — kein Server, keine Installation nötig.

---

## ⌨️ Tastaturkürzel

| Taste        | Aktion         |
|--------------|----------------|
| `Leertaste`  | Würfeln        |
| `R`          | Spiel zurücksetzen |
